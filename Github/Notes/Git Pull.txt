git pull Kyun Better Hai?
Asal mein git pull do (2) commands ka automatic combination hota hai:
git pull = git fetch (Download karna) + git merge (Mix karna)

Agar aap sirf fetch karenge: Toh GitHub wali README aapke system ki hidden memory mein download toh ho jayegi, lekin aapke folder ke andar show nahi hogi. Phir aapko usko merge karne ke liye ek aur command chalani paregi. (Double mehnat).

Agar aap direct pull karenge: Toh yeh ek hi step mein GitHub se us README ko uthayega aur directly aapke local folder ke andar la kar rakh dega.
