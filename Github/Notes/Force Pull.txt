Lekin jab hum git pull ... --allow-unrelated-histories --strategy=subtree lagate hain, toh hum Git ko force kar rahe hote hain ke "Sync mojood nahi hai, lekin tum zabardasti in do alag duniyaon ko aapas mein jor kar ek link bana do."
